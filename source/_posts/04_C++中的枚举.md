# 枚举

枚举值默认从0开始，类内部的枚举并不是一个命名空间，因为每个实例用的枚举都一样，所以可以不构造实例直接通过类进行访问

```C++
#include <iostream>

class Test {
public:
    enum TestEnum {
        D, E = 5, F,G,H=1,I,J,K
    };
};

int main() {

    Test test;

    std::cout << Test::D << Test::E << Test::F << Test::G<< Test::H<< Test::I<< Test::J<< Test::K<< std::endl;
    return 0;
}
```

```
输出为05671234
```
